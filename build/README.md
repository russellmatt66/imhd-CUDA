# Overview
Build folder for `imhd-cuda` and `write-grid`

# Build Instructions
- `$ cmake ../src`
- `$ make clean`
- `$ make`

# Run Instructions
- `$ python3 imhdLauncher.py`
    - Make sure `imhd-cuda.inp` is in the same folder

- `$ python3 gridLauncher.py`
    - Make sure `grid.inp` is in the same folder
